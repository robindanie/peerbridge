# PeerBridge UI/UX Design System (Applied)

This repository now includes a baseline design implementation using the requested palette and an improved multi-select style.

## Files
- `ui/peerbridge-theme.css`: global colors, typography, cards, buttons, links, and input focus styling.
- `ui/multiselect.css`: upgraded multi-select component style with chips and selected-state visuals.

## Color tokens
- Black: `#000000`
- Deep Indigo: `#222052`
- Light Gray: `#B7B7B7`
- Warm Gold / Tan: `#D2B68A`
- Soft Cream: `#EEE5D3`

## Integration snippet

```html
<link rel="stylesheet" href="ui/peerbridge-theme.css" />
<link rel="stylesheet" href="ui/multiselect.css" />
```

Then apply classes like `.pb-btn`, `.pb-card`, `.pb-input`, and `.pb-multiselect` in your UI components.
